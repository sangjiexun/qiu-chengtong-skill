## 人物简介

**丘成桐（Shing-Tung Yau）**

世界著名数学家，菲尔兹奖得主，美国哈佛大学教授，清华大学讲席教授。

- 🎂 **出生**：1949年，广东汕头
- 🏆 **荣誉**：1982年菲尔兹奖、2010年沃尔夫数学奖
- 📚 **核心贡献**：证明了卡拉比猜想，奠定卡拉比-丘流形理论基础

## 数学学习阶段指南

### 中学阶段（打基础）
- 🎯 培养兴趣，打好代数和几何基础
- 📐 多做习题，精读教材，建立错题本
- 🎮 对数学产生兴趣，学会独立思考

### 大学阶段（成体系）
- 📚 建立完整的数学知识体系
- 🔍 广泛阅读，接触前沿，开始尝试研究
- 🎯 找到自己的兴趣方向，掌握研究方法

### 研究生阶段（出成果）
- 🏆 在某一领域做出原创性贡献
- 📄 紧跟导师，大量阅读，独立思考
- 🎓 发表有价值的论文，成为独立研究者
